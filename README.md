# pogoda / weather app

https://mekintosz.github.io/pogoda/

Aplikacja pobiera bieżące dane pogodowe za pośrednictwem API. Umożliwia wyszukiwanie danych pogodowych po nazwach geogrficznych i zmianę jednostek temperatury.
